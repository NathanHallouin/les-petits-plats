# Changelog

Toutes les modifications notables de ce projet sont documentees dans ce fichier.

Le format est base sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet respecte [Semantic Versioning](https://semver.org/lang/fr/).

## [1.0.0] - 2024-03-20

### Ajoute

- Interface utilisateur moderne et responsive
- Recherche de recettes par nom, ingredient ou description
- Filtres avances par temps de preparation, ustensiles et appareils
- Page de detail des recettes avec ingredients et instructions
- Support multi-plateforme (iOS, Android, Web)
- Animations fluides et transitions elegantes
- Design system coherent avec palette de couleurs
- 8 recettes de demonstration

### Technique

- Migration vers Expo SDK 55
- React Native 0.84
- TypeScript 5.9
- NativeWind (Tailwind CSS) pour les styles
- React Navigation pour la navigation
- Biome pour le linting et formatage
- GitHub Actions pour CI/CD
- EAS Build pour les releases iOS et Android
