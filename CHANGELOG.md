# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-20

### Added

- Modern and responsive user interface
- Recipe search by name, ingredient, or description
- Advanced filters by preparation time, utensils, and appliances
- Recipe detail page with ingredients and instructions
- Multi-platform support (iOS, Android, Web)
- Smooth animations and elegant transitions
- Consistent design system with color palette
- 8 demo recipes

### Technical

- Migration to Expo SDK 55
- React Native 0.84
- TypeScript 5.9
- NativeWind (Tailwind CSS) for styling
- React Navigation for navigation
- Biome for linting and formatting
- GitHub Actions for CI/CD
- EAS Build for iOS and Android releases
